# TwiUI
Twilight UI component framework for Bootstrap
TwiUI（中文全称：暮光快速用户界面前端开发框架）是一款以Javascript语言编写的前端UI框架，其目的在于将前端通用的功能框架化，通过参数配置开发方式，降低开发门槛、提高开发效率、建设代码量，实现快速界面构建和开发，1.0版本基于原生态HTML和CSS样式，2.0版本重点基于Bootstrap样式风格构建。 twi 取自英文twilight（中文直译为：暮光）前缀。
TwiUI主要包含的组件有：表格（twiDataTable）、表单（twiForm）、窗体（twiWindow）、消息提示框（twiMsg）、数结构（twiTree）、进度加载蒙版（twiLoadingMask）、模式窗口（twiModalWindow）、富文本编辑器（twiHtmlEditor）、对话框（twiDialog）、确认窗口（twiConfirm）、信息提示工具（twiTooltip）、图片缩放工具（twiPicZoom）、表单验证（twiFormValidate）、下拉列表（twiDropdown）、图片表格（twiImgGrid）、图标（twiIcon）等。
